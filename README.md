# Loss Reserving Model (Chain Ladder Method)

## 📌 Objective
To estimate insurance reserves using historical claims data.

## 🧮 Methodology
- Chain Ladder Method
- Development Factors calculation
- Projection of ultimate claims

## 📊 Key Features
- Automated Excel model
- Inflation-adjusted projections
- Calculation of Best Estimate Liabilities (BEL)

## 🛠 Tools Used
- Microsoft Excel

## 📁 Files
- Excel model

## 🔗 Project Overview
This project demonstrates actuarial reserving techniques using structured loss triangle data.
